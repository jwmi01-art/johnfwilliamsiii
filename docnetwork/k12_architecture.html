Section 01

System Overview

The K12 Retirement Success System™ is a digital retirement planning platform for Michigan K12 educators. It captures prospect data through a lead magnet calculator, delivers personalized results, and automatically provisions a dashboard account — creating a warm lead with data attached before Sean's first contact.

Business Purpose

A prospect visits the lead magnet gap calculator, enters their financial data, and receives a personalized retirement gap report. Their data is automatically saved to the system and a personal dashboard account is created. Sean receives a warm lead with full financial context before any conversation takes place.

Section 12

Making Changes

Adding a New Form Field

  1. Add the HTML input field to K12_Lead_Magnet_Dynamic.html
  2. Add the field to the generateReport() function variable declarations
  3. Add the field to the params URLSearchParams object (for results page)
  4. Add the field to the Make.com webhook JSON body in the fetch() call
  5. Add the field to the Airtable Prospects/Clients table
  6. Update the K12 Webhook Data structure in Make.com JSON module
  7. Map the new field in Make.com Airtable module
  8. Update the Elementor widget on WordPress with the new form code
  9. Optionally add the field to the Softr dashboard block

Adding a New Advisor

  1. Create record in Airtable Advisors table
  2. Create Softr user account for the advisor
  3. Create Advisor user group in Softr
  4. Build advisor portal page in Softr with client-filtered view
  5. Assign existing clients to the new advisor in Airtable

Adding a New District

  1. Create record in Airtable Districts table
  2. Tag relevant client records with the District field
  3. Phase 3: build district HR view in Softr

Updating the Welcome Email

  1. Go to Make.com → Integration Webhooks scenario
  2. Click the SendGrid module (module 7)
  3. Edit the Content field
  4. Use data picker badges for dynamic values (name, magic link)
  5. Save and test with a fresh email address

Updating the Results Page

  1. Edit K12_Results.html locally
  2. Upload to /home/kretirem/public_html/lead_magnet/ via cPanel File Manager
  3. Test by submitting the form and checking the results page
Never do this: Rename Airtable fields that are mapped in Make.com. This silently breaks the automation. If you must rename a field, update the Make.com mapping immediately after.
